## ✨ 特性

- 🌈 支持矩形、圆、多边形的绘制
- 📦 开箱即用的高质量组件
- 🛡 使用 TypeScript 开发，提供完整的类型定义文件。

## 📦 安装

```bash
npm install @canvas-draw/react --save
```

```bash
yarn add @canvas-draw/react
```

```bash
pnpm add @canvas-draw/react
```

```bash
bun add @canvas-draw/react
```
